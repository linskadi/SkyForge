# 传感器融合完整案例

## 概述

本案例展示 SkyForge 如何从自然语言需求自动生成符合 DO-178C/MISRA-C 标准的多传感器融合代码。

## 需求

实现扩展卡尔曼滤波器（EKF），融合加速度计、陀螺仪和磁力计数据：
- 9维状态向量：位置3+速度3+姿态3
- 融合频率：100Hz
- 输出：姿态四元数
- 安全等级：DAL-A

## 生成产物

| 文件 | 说明 |
|------|------|
| `requirement.txt` | 自然语言需求描述 |
| `contract.yaml` | DO-178C合规契约 |
| `expected_code.c` | 生成的MISRA-C代码 |

## 运行方式

```bash
# 使用SkyForge生成
python -c "
from skyforge_engine.pipeline import run_pipeline
import asyncio

result = asyncio.run(run_pipeline('实现一个多传感器数据融合模块，使用卡尔曼滤波'))
print('生成完成')
"
```

## 技术亮点

1. **扩展卡尔曼滤波**：9维状态向量，预测-更新循环
2. **四元数姿态表示**：避免万向锁问题
3. **多传感器融合**：加速度计+陀螺仪+磁力计
4. **MISRA-C合规**：固定大小矩阵运算，无动态内存
5. **故障处理**：传感器失效时降级到纯积分模式
6. **DO-178C追溯**：每个函数/变量标注[REQ-xxx]追溯注释
