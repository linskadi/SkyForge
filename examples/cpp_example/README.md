# C++ 低通滤波器示例

## 概述

本示例展示 SkyForge 如何生成符合 MISRA C++/JSF AV C++ 标准的 C++ 代码。

## 编码规范

- **MISRA C++:2023** - C++安全编码标准
- **JSF AV C++** - 联合攻击战斗机C++编码标准（221条规则）
- **CERT C++** - C++安全编码实践

## 核心规则

1. 禁止使用 new/delete（JSF-008）
2. 禁止使用异常（JSF-006）
3. 禁止使用 dynamic_cast（JSF-005）
4. 禁止使用 RTTI（JSF-007）
5. 使用 RAII 管理资源
6. 所有虚函数使用 override

## 运行方式

`ash
# 编译运行
g++ -std=c++17 -Wall -Wextra -o filter main.cpp
./filter
`
